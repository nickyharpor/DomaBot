# Doma Bot

## Installation
Create a `.env` file and set the `ENV` variable to one of the following:
  - `test`
  - `dev`
  - `live`

Setting the environment tells the bot which config to use:
  - `config_test.py`
  - `config_dev.py`
  - `config_live.py`

Fill in the config file with the appropriate values.
